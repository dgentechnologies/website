# Privacy Notice

This website collects minimal personal data to operate core features.

## Data Collected

- Contact Form: `name`, `email`, `message` stored in Firestore (`contactMessages`).
- Analytics: Aggregate usage metrics (via Vercel Analytics or GA4) without personally identifiable information.

## Usage

- Messages are used to respond to inquiries and improve services.
- Analytics is used to understand site performance and usage.

## Retention

- Contact messages are retained as needed to provide support and for business records.
- You may request deletion via: privacy@dgentechnologies.com.

## Third Parties

- Unsplash for image fetching.
- Google Gemini via Genkit for AI-generated content.

## Security

- Firestore security rules restrict unauthorized writes.
- Secrets are stored in environment variables and not committed to source control.

## Contact

For privacy requests or questions: privacy@dgentechnologies.com.
